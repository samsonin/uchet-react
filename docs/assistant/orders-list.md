# Страница: Все заказы

Маршрут в приложении: `/orders`

Раздел: `Услуги`

## Когда использовать

Используй эту инструкцию, когда пользователь спрашивает:

- как найти заказ;
- где посмотреть список заказов;
- как открыть карточку заказа;
- как показать только свои текущие заказы;
- как скрыть завершенные заказы;
- как искать по дополнительным полям заказа.

## Краткий ответ для пользователя

На странице `/orders` отображается список заказов. Здесь можно быстро найти заказ по основным данным, открыть расширенные параметры поиска, показать свои текущие заказы, скрыть завершенные и перейти в карточку нужного заказа.

## Что можно сделать на странице

1. Найти заказ через основную строку поиска. Можно искать по номеру заказа, телефону, фамилии и имени клиента, а также по наименованию или серийному номеру устройства. Введите запрос и нажмите `Найти` или клавишу `Enter`.
2. Открыть `Параметры поиска`, если нужно искать по специальным или дополнительным полям заказа.
3. Нажать `Мои текущие`, чтобы быстро отобразить заказы, которые сейчас относятся к текущему пользователю.
4. Включить `скрыть завершенные`, чтобы в списке остались только незакрытые заказы.
5. Кликнуть по строке заказа, чтобы открыть его карточку в новой вкладке.

## Подсказки для ассистента

- Если пользователь уже находится на `/orders`, не предлагай ему открыть эту же страницу.
- Если пользователь спрашивает про поиск, сначала объясни основную строку поиска, затем `Параметры поиска`.
- Если пользователь не видит нужный заказ, предложи проверить, не включен ли фильтр `скрыть завершенные`, и уточнить поисковый запрос.
- Если пользователь хочет посмотреть только свои активные работы, направь его к кнопке `Мои текущие`.
- Не обещай, что поиск найдет заказ по любому произвольному тексту: перечисляй поддерживаемые поля поиска.

## Пример ответа ассистента

На странице `Все заказы` вы можете найти нужный заказ через строку поиска: введите номер заказа, телефон, имя или фамилию клиента, наименование либо серийный номер устройства и нажмите `Найти` или `Enter`. Для поиска по дополнительным полям откройте `Параметры поиска`. Если нужны только ваши текущие заказы, нажмите `Мои текущие`; чтобы убрать завершенные, включите `скрыть завершенные`. Карточка заказа открывается кликом по строке заказа.
