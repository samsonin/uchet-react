# Страница: Способы оплаты

Маршрут в приложении: `/settings/payment-types`

## Когда использовать

Используй эту инструкцию, когда пользователь спрашивает:

- где настроить способы оплаты;
- как добавить новый способ оплаты;
- как изменить способ оплаты;
- как удалить способ оплаты;
- где управляются способы оплаты организации.

## Краткий ответ для пользователя

В разделе `/settings/payment-types` вы можете управлять способами оплаты, которые используются в вашей организации: добавлять новые варианты, редактировать существующие и удалять те, которые больше не нужны.

## Что можно сделать на странице

1. Добавить новый способ оплаты для организации.
2. Отредактировать название или параметры уже созданного способа оплаты.
3. Удалить способ оплаты, если он больше не используется.
4. Просмотреть список всех способов оплаты, доступных в организации.

## Подсказки для ассистента

- Если пользователь пишет `/payment-types`, уточняй правильный маршрут: `/settings/payment-types`.
- Если пользователь уже находится на `/settings/payment-types`, не предлагай ему открыть эту же страницу.
- Объясняй, что эти способы оплаты используются внутри организации при работе с платежами.

## Пример ответа ассистента

В разделе `Способы оплаты` можно добавить, изменить или удалить варианты оплаты, которые используются в вашей организации. Откройте `/settings/payment-types`, чтобы увидеть текущий список способов оплаты и настроить его под ваш рабочий процесс.
