# Страница: Физические лица

Маршрут в приложении: `/customers`

## Когда использовать

Используй эту инструкцию, когда пользователь спрашивает:

- где найти клиента;
- как найти физическое лицо;
- где находится база клиентов;
- как искать клиента по телефону, имени, документам, адресу или дополнительным контактам;
- как открыть карточку клиента.

## Краткий ответ для пользователя

Клиентов нужно искать на странице `/customers`, в разделе `Физические лица`. Перейдите в этот раздел и воспользуйтесь поиском по списку клиентов.

## Что можно сделать на странице

1. Найти физическое лицо через строку поиска.
2. Искать клиента по имени, телефону, документам, адресу, e-mail, заметкам и дополнительным контактам.
3. Открыть карточку клиента кликом по строке клиента.
4. Добавить нового клиента через кнопку добавления.
5. Быстро перейти к контактам клиента: телефон, e-mail, Telegram, WhatsApp, Instagram, сайт или другой дополнительный контакт, если они заполнены.

## Подсказки для ассистента

- На вопрос `Где найти клиента?` направляй пользователя на `/customers`, а не на `/orders`.
- Важно отметить, что нужен раздел `Физические лица`.
- Если пользователь уже находится на `/customers`, не предлагай открыть эту же страницу: скажи, что можно воспользоваться поиском по разделу.
- Не отправляй пользователя в `Все заказы` для поиска клиента. В заказах можно встретить данные клиента, но основная база клиентов находится в `/customers`.

## Пример ответа ассистента

Клиентов нужно искать в разделе `Физические лица` на странице `/customers`. Перейдите туда и воспользуйтесь поиском по разделу: можно искать по имени, телефону, документам, адресу и дополнительным контактам. Карточка клиента открывается кликом по строке клиента.
